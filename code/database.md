# Databse.py
Данный файл отвечает за создание базы данных и её наполнение.  
Для данного проекта была выбрана библиотека sqlite3, т.к она хорошо подходит для создания небольших проектов и достаточно легка в использовании.
## Соединение с БД
```
conn = sqlite3.connect("furniture.db")
cursor = conn.cursor()
```
Отвечает за подключение базы данных с названием `furniture.db` к проекту и создание курсора для работы с ней в дальнейшем. Если файла с БД не существует, то он его автоматически создаёт.
## Создание таблицы
```
cursor.execute("""
CREATE TABLE IF NOT EXISTS furniture(
               id INTEGER PRIMARY KEY AUTOINCREMENT,
               name TEXT NOT NULL,
               category TEXT,
               price INTEGER,
               material TEXT,
               size TEXT,
               color TEXT,
               available INTEGER CHECK (available IN (0, 1)) DEFAULT 1
               )
""")
```
Если подобной таблицы не существует, то создаёт её с указанными столбцами.

## Наполнение БД данными
```
cursor.executemany("INSERT INTO furniture(name, category, price, material, size, color, available) VALUES (?, ?, ?, ?, ?, ?, ?)",
               catalog)
```
Вносит данные из списка `catalog` в таблицу `furniture`.
## Подтверждение изменений
```
conn.commit()
conn.close()
```
Сохраняет изменения и закрывает соединения с базой данных.
