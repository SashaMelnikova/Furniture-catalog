# Furniture-catalog
Маленькое простое приложение, которое позволяет просматривать товары каталога и фильтровать их по необходимости.
![Каталог мебели](./images/example.png)
# Основные функции
* Просмотр каталога мебели
* Фильтрация каталога по как-либо параметрам  
Например фильтрация по диапазону цен:
